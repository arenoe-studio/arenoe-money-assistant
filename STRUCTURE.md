## Key Directories

- **src/bot**: Where the bot interaction logic lives. `scenes` manage complex multi-step conversations.
- **src/services**: Core business logic. Separated from bot interface for easier testing and cleaner code.
- **src/db**: Database definition using Drizzle ORM.
- **drizzle**: SQL migration files generated by Drizzle Kit. DO NOT edit manually.
